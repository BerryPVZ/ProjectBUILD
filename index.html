Project: BUILD 4.0

Project: BUILD 4.0 sandbox. Mockup-style packet UI, level tools, custom packets, multi-projectiles, and broader PVZ1 recreation abilities while keeping the full editor stack.

Tool

Placement Grid

Configure the grid, preview it, then initialize it. Once initialized, the grid hides. Plants snap to it; zombies, bosses, NPCs, and props do not.

Tile Painter

Paint special tile rules on the grid. Pool tiles require Aquatic plants. Roof tiles require Tilegrown plants.

Main Lawn

By default, this tries to load main.png and main.mp3 from the same folder as this HTML file. You can still override them below.

Template Maker

Normal plants can only be planted on land tiles. Aquatic plants can be planted on pool tiles. Tilegrown plants can be planted on roof tiles.

3.0 Tuning

3.0 fields save into templates and worlds. Old worlds still load with safe defaults.

NPCs do not use abilities. They can play a click sound and show multi-slide dialog when clicked in Select mode.

Templates

Wave Scripter / Level Editor

Uses your saved zombie/boss templates as the zombie library. Start Level shows READY, SET, PREPARE! before enemies spawn.

World

Project: BUILD controls: click to place. Plants snap to the initialized grid; zombies do not. Select mode lets you drag, click NPCs for dialog, mouse wheel to scale, Shift + wheel to rotate, Delete to remove selected.

Tile Paint Mode
